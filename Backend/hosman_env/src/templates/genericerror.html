Error 400-ish Something... ;(

You have requested something that the API server doesn't know how to handle. To resolve this issue, please refer to the documentation.


Some possible errors:-

  • 404 - Resource not found in the server
  • 403 - Access denied
  • 405 - Method not allowed
  • 410 - Gone (This response is sent when the requested content has been permanently deleted from server, with no forwarding address)
